IPNOVA VPN - Windows install package
====================================

Extract the ZIP so you see these files on ONE folder (not nested twice):
  Install IPNOVA.bat
  Setup-IPNOVA.ps1
  README.txt

Customer: double-click  Install IPNOVA.bat

If error "application files are missing":
  Run build-windows-installer.ps1 on Windows (Node + Rust), then use the new ZIP.

Windows 10 (1809+) or 11, 64-bit.
